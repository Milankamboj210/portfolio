# 💼 Milan Kamboj - Personal Portfolio

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, projects, and journey as a Computer Science student.

## 🌐 Live Demo

🔗 **Portfolio:** https://myportfolio-five-sigma-35.vercel.app/

---

## 🚀 Features

- 🎨 Modern dark theme UI
- 📱 Fully responsive design
- ✨ Smooth ScrollReveal animations
- ⌨️ Typing animation in Hero section
- 🖼️ Interactive 3D profile image effect
- 📂 Projects showcase section
- 🛠️ Technical Skills section
- 📄 Resume download button
- 📬 Contact form
- 🔗 Social media links
- ⚡ Fast deployment using Vercel

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- ScrollReveal.js
- Typed.js
- Font Awesome
